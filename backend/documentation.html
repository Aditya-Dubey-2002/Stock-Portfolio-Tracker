StockItUp API Documentation

Base URL

  • Production: https://stock-portfolio-tracker-2oud.onrender.com
  • Local: http://localhost:5000

Authentication Routes

Register User

POST /api/auth/register

  • Request Body:
{
  "fullName": "John Doe",
  "email": "john@example.com",
  "password": "securePassword123"
}
  • Response:
    • 201 Created:
      {
  "token": "<JWT Token>"
}
    • 400 Bad Request:
      {
  "message": "User already exists"
}
    • 500 Internal Server Error:
      {
  "message": "Server error"
}

Login User

POST /api/auth/login

  • Request Body:
{
  "email": "john@example.com",
  "password": "securePassword123"
}
  • Response:
    • 200 OK:
      {
  "token": "<JWT Token>"
}
    • 400 Bad Request:
      {
  "message": "Invalid credentials"
}
    • 500 Internal Server Error:
      {
  "message": "Server error"
}

User Routes

Get User Profile

GET /api/user/profile

  • Headers:
    • Authorization: Bearer <JWT Token>
  • Response:
    • 200 OK:
      {
  "userDetails": {
    "fullName": "John Doe",
    "email": "john@example.com",
    "bio": "Investor and trader",
    "balance": 10000.50,
    "image": "<Profile Image URL>"
  }
}
    • 401 Unauthorized:
      {
  "error": "Access denied, token missing"
}
    • 404 Not Found:
      {
  "error": "User not found"
}

Order Routes

Create Order

POST /api/orders

  • Headers:
    • Authorization: Bearer <JWT Token>
  • Request Body:
{
  "stockId": "AAPL",
  "orderType": "buy",
  "price": 150.00,
  "quantity": 5
}
  • Response:
    • 201 Created:
      {
  "message": "Buy order placed successfully",
  "order": {
    "orderId": 1,
    "userId": 101,
    "stockId": "AAPL",
    "orderType": "buy",
    "quantity": 5,
    "amount": 750.00
  }
}
    • 400 Bad Request:
      {
  "message": "Insufficient balance"
}
    • 500 Internal Server Error:
      {
  "message": "Transaction failed after multiple retries"
}

Get Orders

GET /api/orders

  • Headers:
    • Authorization: Bearer <JWT Token>
  • Response:
    • 200 OK:
      [
  {
    "orderId": 1,
    "stockId": "AAPL",
    "orderType": "buy",
    "quantity": 5,
    "amount": 750.00
  }
]
    • 404 Not Found:
      {
  "message": "No orders found"
}
    • 500 Internal Server Error:
      {
  "message": "Failed to fetch orders"
}
